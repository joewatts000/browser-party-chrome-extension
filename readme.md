# Browser party extension

Covers the current webpage with animations:

- fireworks
- explosions
- sparkles
- party mode (all)

`bun install` - installs packages

`bun run dev`
`bun run build`
`bun run lint`
`bun run lint:fix`
`bun run format`

edit the config in `./src/utils/effects.js` to change the animations
